## Typora常用功能快捷键

| Bold           | **Text**                                | Ctrl + B         |
| -------------- | --------------------------------------- | ---------------- |
| Emphasize      | *text*                                  | Ctrl + I         |
| Strike-through | ~~text~~                                |                  |
| Link           | [title](https://typora.io/#)            | Ctrl + K         |
| Inline Code    |                                         | Ctrl + Shift + K |
| Image          | ![](F:\清远基地项目\客户端\Setting.png) | Ctrl + Shift + I |
| List           | * item                                  |                  |
| Blockquote     | > quote                                 |                  |
| H1             | # text                                  |                  |
| H2             | ## text                                 |                  |
| Table          |                                         | Ctrl + T         |
| 公式           | $$ + enter                              |                  |
